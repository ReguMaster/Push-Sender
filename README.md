# Push-Sender

소개
=============
### Push-Sender 는 Firebase 기반으로 푸시 메세지를 편하게 보낼 수 있게 도와줍니다.
##### 간단한 쿼리문으로 푸시 메세지를 간편하게 보낼 수 있습니다.

* 큐(Queue) 기반 푸시 대기열 시스템
* Google Firebase 를 사용하여 비용 없음
* 로그 테이블 저장을 통해 월별 통계 기능
